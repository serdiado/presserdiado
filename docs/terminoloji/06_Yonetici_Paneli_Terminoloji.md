\# Presserdiado Yönetici Paneli Terminoloji Sözlüğü



Bu dosya, Presserdiado yönetici panelinde kullanılacak terimleri tanımlar.



Bu bölüm; platform yöneticisi, operasyon ekibi, matbaa operatörü, destek ekibi, finans ekibi ve sistem yöneticilerinin kullanacağı yönetim ekranlarının dilini kapsar.



Bu dosya iki parça halinde hazırlanmıştır:



\- 1. Parça: 1–7. maddeler

\- 2. Parça: 8–14. maddeler



\---



\## Kullanım Kuralı



\- Kullanıcı arayüzünde \*\*Önerilen Terim\*\* alanı esas alınır.

\- İngilizce / teknik karşılıklar yalnızca geliştirici dokümantasyonu, veri modeli veya teknik açıklamalarda kullanılabilir.

\- Yönetici panelinde kullanıcı paneline göre daha teknik terimler kullanılabilir; ancak yine de Türkçe karşılıklar tercih edilmelidir.

\- “Dashboard” yerine yönetici panelinde \*\*Kontrol Paneli\*\* kullanılmalıdır.

\- “Job Ticket” yerine \*\*İş Emri\*\* veya \*\*Üretim Fişi\*\* kullanılmalıdır.

\- “Preflight” yerine \*\*Baskı Kontrolü\*\* kullanılmalıdır.

\- “Workflow” yerine \*\*İş Akışı\*\* kullanılmalıdır.

\- “Fulfillment” yerine bağlama göre \*\*Üretim / Hazırlık / Sevkiyat\*\* kullanılmalıdır.

\- “Billing” yerine \*\*Fatura / Ödeme / Muhasebe\*\* kullanılmalıdır.

\- “Analytics” yerine \*\*Raporlama / Analitik\*\* kullanılabilir.




Yönetici panelinde teknik terimler kullanıcı paneline göre daha fazla kullanılabilir; ancak İngilizce terimler doğrudan ana menü, buton ve uyarı metinlerinde kullanılmamalıdır.



Özellikle şu dönüşümler korunmalıdır:



```text

Dashboard → Kontrol Paneli

Admin Panel → Yönetici Paneli

Job Ticket → İş Emri

Workflow → İş Akışı

Preflight Report → Baskı Kontrol Raporu

Ticket → Destek Talebi

Permission → Yetki

Role → Rol

Audit Log → Denetim Kaydı

Queue → Kuyruk / İş Kuyruğu

```



Bu dosya, yönetici panelindeki menü adları, tablo başlıkları, durum etiketleri, buton metinleri ve operasyon açıklamaları için ana terminoloji kaynağıdır.
